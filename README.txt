How to Use:
1. Install the source code into a directory and compile by typing: g++ etc..
2. To run at the command line, type: ./isostocks
3. Enter your account balance
4. If you type 2, you get to see the list of stocks available
5. Type 3 to begin the game
6. Select the buy option
7. Enter the amount of days you wish to advance
8. If you show you bank account balance you should be able to show the difference in price
9. sell the stock you simply select the number position and you sell it
10. Rinse, wash, and repeat.
